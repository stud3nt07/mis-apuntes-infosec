PLAN DE ESTUDIO: Linux – De Básico a Idóneo (8 Semanas)
Semana 1 – Fundamentos del sistema
Objetivos:
- Manejar la terminal con soltura.
- Entender la estructura del sistema de archivos Linux.
Contenido:
- Comandos básicos: ls, cd, cat, cp, mv, rm, less, file, touch.
- Búsqueda avanzada: find, locate, grep.
- Estructura de directorios: /etc, /var, /usr, /home, /tmp.
Ejercicios:
- Navegar solo con terminal por 2 días.
- Buscar archivos:
find / -type f -name "archivo*"
- Crear, mover y borrar archivos.
Semana 2 – Usuarios, grupos y permisos
Objetivos:
- Dominar permisos de archivos.
- Administrar usuarios y grupos.
Contenido:
- chmod, chown, chgrp.
- Permisos especiales: SUID, SGID, Sticky Bit.
- useradd, passwd, groupadd.
Ejercicios:
- Crear usuarios y grupos.
- Asignar permisos.
- Configurar sticky bit en /tmp.
Semana 3 – Gestión de paquetes
Contenido:- Fedora: dnf install/remove/history.
- OpenSUSE: zypper in/up/dup.
Ejercicios:
- Instalar y borrar software.
- Agregar repositorios.
- Revisar historial.
Semana 4 – Servicios, procesos y logs
Contenido:
- systemctl start/stop/status.
- journalctl -xe.
- ps, top, htop, kill.
Ejercicios:
- Habilitar servicios.
- Revisar logs.
- Identificar procesos pesados.
Semana 5 – Redes
Contenido:
- ip a, ip r, ip link.
- nmcli.
- DNS, DHCP, rutas.
Ejercicios:
- Cambiar IP manual.
- Reiniciar red.
- traceroute.
Semana 6 – Shell Scripting
Contenido:
- Variables, loops, condiciones.- Funciones.
Ejercicios:
- Script de monitoreo de disco.
- Script de backups automáticos.
Semana 7 – Seguridad y firewall
Contenido:
- firewalld, ufw.
- iptables básico.
Ejercicios:
- Abrir puertos.
- Revisar reglas.
- Bloquear IPs.
Semana 8 – Virtualización y contenedores
Contenido:
- KVM/QEMU/virt-manager.
- Docker/Podman.
Ejercicios:
- Crear VMs.
- Ejecutar servidor en contenedor.
- Crear imágenes.
Rutina Semanal:
Lunes a viernes: 45 min diarios.
Sábado: 1 hora de práctica.
Domingo: descanso.
Resultado esperado:
Administración completa de Linux, redes, servicios, scripting, seguridad y contenedores.
